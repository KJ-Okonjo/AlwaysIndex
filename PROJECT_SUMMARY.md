## 💡 Design Philosophy

This website is built with these principles:

1. **Execution over Theory**: Production-ready, not abstract concepts
2. **Clarity over Complexity**: Clean structure, easy to navigate
3. **Culturally Aligned**: Editorial tone with professional credibility
4. **Mobile-First**: Majority of users will access on mobile
5. **Conversion-Focused**: Clear CTAs and value propositions
6. **No Friction**: WhatsApp/Email only—no forms, no checkout
